
교재에는 자세한 설명이 부족하기 때문에, 내가 필요하다고 느낀 개념들은 여기에 정리한다.

추가적으로, 교재에 설명이 챕터에 맞지 않게 나온 부분도 여기에 링크한다.

# 일반적인 학습 과정

1. Forward Pass : Input Data → Model → Prediction
2. Loss Calculation
3. Backward Pass : Update gradients
4. Optimizer Step : Update weights
5. Zeroing Gradients : Set gradients to zero
