# 개요
이미지는 <높이, 너비, 채널> 이라는 3차원 텐서이다.

# 흑백 이미지
흑백 이미지는 채널 수가 1이다.
![[28x28 handwriting data.png]]

# 컬러 이미지
컬러 이미지는 채널 수가 3이다.
![[Image RGB Channel.png]]
예를 들어. 28x28이미지라면 텐서의 모양은(28, 28, 3)이다.
이때 각 이름은 (높이, 너비, 깊이=채널)이라고 한다.

# 표시 형식
Numpy 나 보통의 라이브러리에서는 (높이, 너비, 채널) 형태를 사용하지만
PyTorch에서는 (채널, 높이, 너비)를 사용한다.
