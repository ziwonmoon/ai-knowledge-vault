# 개요
선형 변환과 비선형 [[Activation Function|활성화 함수]]를 여러 층으로 조합한 함수 모델

로지스틱 모델이 "선형 + 하나의 비선형"이라면 뉴럴 네트워크는 비선형 변환을 여러 층으로 확장한 모델

복잡한 함수 근사가 가능해진다.
